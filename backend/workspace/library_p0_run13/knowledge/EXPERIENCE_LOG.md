# 产品需求文档 (PRD) — 图书管理系统

## 1. 项目概述

开发一套图书管理系统，实现图书的借阅、归还、读者管理以及查询统计等核心业务流程，提升图书管理效率与用户体验。系统采用 B/S 架构，后端基于 Python FastAPI 构建 RESTful API，前端提供简洁易用的 Web 界面。

## 2. 用户角色

| 角色 | 描述 | 主要职责 |
|------|------|----------|
| 管理员 | 系统管理人员 | 录入/编辑/下架图书；管理读者信息；查看统计数据；处理异常借还 |
| 读者 | 图书借阅者 | 查询图书；借阅/归还图书；查看个人借阅记录 |

## 3. 核心功能列表

### 3.1 图书管理
- **图书录入**：管理员新增图书，包括书名、作者、ISBN、出版社、出版年份、分类、库存数量等字段
- **图书编辑**：管理员修改图书信息
